## Assignment 1
Tested on ubuntu 20.04

### Install dependencies
`sudo apt-get install build-essential`

### Compiling the program(s)
`make compile_tcp`  
`make compile_udp`  

### Running the TCP program(s)
open new terminal and run:  
`./tcp_server`  

open other terminal:  
`./tcp_client`  

### Running the UDP program(s)
open new terminal and run:  
`./udpserver`  

open other terminal:  
`./udpclient`
